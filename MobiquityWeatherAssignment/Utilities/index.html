Home Screen

In The "HomeViewController" List of all Selected Locations appears. And at the bottom of the page "Add location" button present on tap of that button Mapview will open. In MapView search Box user can search any location and can add any location to the map.".

Key features

  1. Once the desired location is added user will be navigated back to HomeViewcontroller where he can see all the added Location in the Tableview
  2. On tap of any location from the HomeViewcontroller page user will be taken to Forecasting page .Where I have used a Table-StackView combination to show all the forecasting data of particular location.
  3. In SettingsViewcontroller User can reset all the Bookmarks on the Mapview,And user can delete all the locations which previously added to the Tableview in HomeViewcontroller.
  4. Compnents used are 1.Stackview &Tableview. 2.CoreData to store location info 3.Weather API services to fetch forecasting data.
